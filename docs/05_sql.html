SQL

SQL (анг. Structured query language) - мова запитів на якому ми спілкуємось з базами даних.

База даних - це сховище даних, де зберігаються наші “таблиці”.

Схема бази даних

Типи даних

В таблицях можуть зберігатися різні типи даних: цілі і дробові числа, текст, дати, масиви чисел. У цих даних ви зустрінетесь з наступними типами:

Типи даних Опис Приклад
INT Ціле число id користувача: 123
NUMERIC / DECIMAL Дійсне число Вартість товару: 120.55
VARCHAR Текст Дія із замовленням: «create_order»
DATE Дата з точністю до дня Дата народження користувача: 25/03/91
TIMESTAMP Дата з точністю до секунди Час реєстрації у додатку: 24/08/22 01:52:24
[] Масив Список id товаров у замовленні: [1, 13, 22]

Більш детально почитати про типи даних можна за посиланням

Структура та наповнення таблиць

user_actions – дії користувачів із замовленнями.

Стовпчик Тип даних Опис
user_id INT id користувача
order_id INT id замовлення
action VARCHAR(50) дія користувача із замовленням; “create_order” - створення замовлення, “cancel_order” - скасування замовлення
time TIMESTAMP час дії

courier_actions – дії кур’єрів із замовленнями.

Стовпчик Тип даних Опис
courier_id INT id кур’єра
order_id INT id замовлення
action VARCHAR(50) дія кур’єра із замовленням; ‘accept order’ - прийняття замовлення, ‘deliver order’ - доставка замовлення
time TIMESTAMP час дії

orders - інформація про замовлення.

Стовпчик Тип даних Опис
order_id INT id замовлення
creation_time TIMESTAMP час створення замовлення
product_ids integer[] список id товарів у замовленні

users - інформація про користувачів.

Стовпчик Тип даних Опис
user_id INT id користувача
birth_date DATE дата народження
sex VARCHAR(50) стать

couriers - інформація про кур’єрів.

Стовпчик Тип даних Опис
courier_id INT id кур’єра
birth_date DATE дата народження
sex VARCHAR(50) стать

products - інформація про товари, які доставляє сервіс.

Стовпчик Тип даних Опис
product_id INT id продукту
name VARCHAR(50) назва товару
price FLOAT(4) ціна товару
Примітка

У дужках типу VARCHAR вказано максимально допустиму кількість символів у тексті. У типу даних NUMERIC у дужках вказано загальну кількість символів.
