SQL

SQL (анг. Structured query language) - мова запитів на якому ми спілкуємось з базами даних.

База даних - це сховище даних, де зберігаються наші “таблиці”.

Схема бази даних

У цьому розділі ми будемо використовувати набори даних, які представляють сервіс з доставки продуктів. Тут зберігається документація, котра допоможе розібратися з ними. У таблиці продемонстровані зв’язки між таблицями, а також опис даних.

Типи даних

В таблицях можуть зберігатися різні типи даних: цілі і дробові числа, текст, дати, масиви чисел. У цих даних ви зустрінетесь з наступними типами:

         Типи даних                       Опис
1               INT                 Ціле число
2 NUMERIC / DECIMAL               Дійсне число
3           VARCHAR                      Текст
4              DATE     Дата з точністю до дня
5         TIMESTAMP Дата з точністю до секунди
6                []                      Масив
                                      Приклад
1                         id користувача: 123
2                    Вартість товару: 120.55 
3          Дія із замовленням: «create_order»
4       Дата народження користувача: 25/03/91
5 Час реєстрації у додатку: 24/08/22 01:52:24
6 Список id товаров у замовленні: [1, 13, 22]

Більш детально почитати про типи даних можна за посиланням

Структура та наповнення таблиць

user_actions – дії користувачів із замовленнями.

  Стовпчик   Тип даних
1  user_id         INT
2 order_id         INT
3   action VARCHAR(50)
4     time   TIMESTAMP
                                                                                                           Опис
1                                                                                                id користувача
2                                                                                                 id замовлення
3 дія користувача із замовленням; "create_order" - створення замовлення, "cancel_order" - скасування замовлення
4                                                                                                       час дії

courier_actions – дії кур’єрів із замовленнями.

    Стовпчик   Тип даних
1 courier_id         INT
2   order_id         INT
3     action VARCHAR(50)
4       time   TIMESTAMP
                                                                                                      Опис
1                                                                                               id кур'єра
2                                                                                            id замовлення
3 дія кур'єра із замовленням; 'accept order' - прийняття замовлення, 'deliver order' - доставка замовлення
4                                                                                                  час дії

orders - інформація про замовлення.

       Стовпчик Тип даних                           Опис
1      order_id       INT                  id замовлення
2 creation_time TIMESTAMP       час створення замовлення
3   product_ids integer[] список id товарів у замовленні

users - інформація про користувачів.

    Стовпчик   Тип даних            Опис
1    user_id         INT  id користувача
2 birth_date        DATE дата народження
3        sex VARCHAR(50)           стать

couriers - інформація про кур’єрів.

    Стовпчик   Тип даних            Опис
1 courier_id         INT      id кур'єра
2 birth_date        DATE дата народження
3        sex VARCHAR(50)           стать

products - інформація про товари, які доставляє сервіс.

    Стовпчик   Тип даних         Опис
1 product_id         INT  id продукту
2       name VARCHAR(50) назва товару
3      price  \tFLOAT(4)  ціна товару
Примітка

У дужках типу VARCHAR вказано максимально допустиму кількість символів у тексті. У типу даних NUMERIC у дужках вказано загальну кількість символів.
